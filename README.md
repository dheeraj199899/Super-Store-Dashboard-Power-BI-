# 📊 Super Store Dashboard (Power BI)

### 🔎 Project Overview
This project demonstrates how raw CSV data can be transformed into **business insights** using **Power BI only**.  
The dashboard provides a clear view of **sales, profit, orders, discounts, and customer trends** across different years, categories, and regions.

As a data analyst aspirant, this project highlights my ability to:
- **Source**: Data downloaded from Kaggle — *Super Store Sales Data*  
- Import and clean raw data  
- Apply transformations in Power Query  
- Create KPIs using DAX functions  
- Design interactive dashboards with filters, slicers, and visuals  

---

### 🧹 Data Preparation (Power Query)
Steps performed in **Power Query**:
- Standardized date format from text → proper date type  
- Removed errors and invalid records  
- Dropped unnecessary columns not required for analysis  
- Ensured clean, structured data ready for visualization  

---

### 📊 Metrics & Calculations (DAX)
Created custom measures using **DAX functions** for KPIs:
- **Total Orders**  
- **Total Profit**  
- **Total Sales**  
- **Profit Margin %**  

These metrics drive the summary cards and provide quick business insights.

---

### 📈 Dashboard Features
The dashboard includes multiple visuals to analyze performance:

#### 🔹 KPIs
- Total Orders: **2K**  
- Total Profit: **127.87K**  
- Total Sales: **507.79K**  
- Profit Margin: **25%**  

#### 🔹 Visuals
- **Line Chart**: Sales & Profit by Year (2014–2017)  
- **Bar Chart**: Profit by Sub-Category (Copiers, Machines, Phones, Chairs, etc.)  
- **Map**: Sales by State (geographical distribution)  
- **Scatter Plot**: Discount vs Profit vs Sales  
- **Customer Analysis**: Sales by Customer ID  

#### 🔹 Interactivity
- Applied **filters & slicers** for Region and Category  
- Enabled drill-down for detailed analysis  

---

### 🚀 Key Insights
- **Seasonality**: Q4 shows consistent peaks in sales  
- **Category Performance**: Copiers & Machines drive the highest profit margins  
- **Regional Trends**: Central & West regions contribute significantly to revenue  
- **Discount Impact**: Higher discounts reduce profitability
